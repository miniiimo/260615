# 🕵️‍♂️ 우리 반 AI 로봇의 비밀 (도서관 방탈출)

## 🎯 학습 목표

2. 공평하고 올바른 AI를 만들기 위해서는 다양하고 균형 잡힌 데이터가 필요함을 깨닫는다.
3. 방탈출 미션을 통해 주도적으로 문제를 해결한다.
